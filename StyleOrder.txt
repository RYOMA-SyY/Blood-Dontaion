LEADERBOARD PAGE STYLE SPECIFICATIONS
====================================

1. NAVBAR STYLES
---------------
- Logo: 40px height
- Nav links: Right-aligned (ms-auto)
- Profile button: 40x40px circular avatar
- Default profile picture: "assets/img/unknown_profile.png"

2. LEADERBOARD TABLE STYLES
--------------------------
- User rows:
  * Avatar: 40x40px with 50% border-radius
  * Online badge: Small green dot indicator
  * Donor info: Name and badge display
  * Donations count: Number display
  * Points: Number with "pts" label

3. RANK INDICATORS
-----------------
- Gold rank: Crown icon (bx bxs-crown)
- Silver rank: Medal icon (bx bxs-medal)
- Bronze rank: Medal icon (bx bxs-medal)

4. STATISTICS SECTION
--------------------
- Stat cards with icons:
  * Blood donation icon (bx bx-donate-blood)
  * Group icon (bx bx-group)
  * Heart icon (bx bx-heart)
- Trend indicators with up arrows (bx bx-up-arrow-alt)

5. ACHIEVEMENT CARDS
-------------------
- Icons:
  * Trophy (bx bxs-trophy)
  * Heart (bx bxs-heart)
  * Star (bx bxs-star)
- Progress bars with percentage indicators

6. EVENT CARDS
-------------
- Date display with day and month
- Location icon (bx bxs-map)
- Time icon (bx bxs-time)
- User count icon (bx bxs-user-plus)
- Urgent badge for emergency events

7. COLOR SCHEME
--------------
- Theme color: #0D0D0D
- Text colors: White for main text
- Accent colors for different ranks (gold, silver, bronze)
- Green for positive trends
- Red for urgent events

8. TYPOGRAPHY
------------
- Font family: Manrope (400 and 700 weights)
- Font Awesome icons (v5.10.0)
- Boxicons (v2.1.4)

9. LAYOUT
---------
- Bootstrap 5.3.0 framework
- Responsive design
- Container-fluid for full-width sections
- Grid system for stats and achievements

10. INTERACTIVE ELEMENTS
-----------------------
- Hover effects on buttons and cards
- Dropdown menu for profile
- Toggle buttons for scope selection
- Register buttons for events

11. FOOTER
----------
- Timestamp display
- Minimal design

12. MOBILE CONSIDERATIONS
------------------------
- Collapsible navbar
- Responsive table layout
- Stacked cards on smaller screens

13. ADDITIONAL FEATURES
----------------------
- Back button navigation
- Call-to-action banner
- Progress indicators
- Badge system for donor status

IMPORTANT NOTES:
- All images should be optimized for web
- Ensure responsive design works on all screen sizes
- Maintain consistent spacing and alignment
- Test all interactive elements
- Verify color contrast for accessibility
